# Changelog

All notable changes to this project will be documented in this file.

## [2.3.1] - 2026-06-12

### Fixed
- Fixed an issue where the update option did not load the hearing log details for previously added logs. It now properly loads and edits the existing hearing record.
- Fixed hearing agenda card background visibility in dark mode on the Dashboard.

### Added
- Added a native date-time picker for the hearing date field to improve user experience.

### Changed
- Dynamic color changes applied for tabs (SegmentedButtons) and status ticks according to the hearing stages.
